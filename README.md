# MyBaseProject
Android常用工具类及集锦以及Mvp框架Base封装<br> 
只需导入common module方可开启快速开发。常见效果会以demo形式展示,持续更新中..  [Demo下载点我](https://github.com/Dalanger/MyBaseProject/blob/dalang/demo.apk)

4.仿淘宝/京东地址选择 <br>
